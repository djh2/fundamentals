Goals

The goals of this lab are to learn how to correctly override the equals method in Java. To do so, we need to learn a bit about hash tables and the HashCode method. We use this to introduce the commercially used test harness, JUnit.

Try to get through all of the lab material. If you have time at the end, start looking at the homework assignments and ask questions if you do not understand something.
